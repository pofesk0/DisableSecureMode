# DisableSecureMode

Идея блокировки использования безопасного режима на Android

И так:

Устанавливаем пароль на телефон и стороннюю клавиатуру. Включаем её в настройках клавиатур и там же отключаем системную, если можно отключить, если нет, просто переключаемся на стороннюю (делаем клавиатурой по умолчанию) и идем к следующему шагу. Загружаемся в безопасный режим и смотрим: появилась ли клавиатура для ввода пароля, если нет - блокировка успешна, если да, отключаем системное приложение клавиатуры через ADB (adb shell pm disable-user --user 0 ИМЯ.ПАКЕТА)
Готово.

В обычном режиме будет работать сторонняя клавиатура и можно ввести пароль.
